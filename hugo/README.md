## Cheat Sheet

> Hugo is one of the most popular open-source static site generators. It makes
> building websites fun again.

### Create a new site

```sh
# create a new site
hugo new site ./blog.example.com
```

```sh
# compile a site
hugo
```

```sh
# serve a site in dev mode
hugo server -D
```
